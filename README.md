üöÄ Installation (Developer Mode)
FOHKASU is currently in Beta. To install it on Chrome/Edge/Brave:
Download this repository (Code -> Download ZIP) and unzip it.
Open your browser and go to chrome://extensions.
Enable Developer Mode (toggle in the top right).
Click Load Unpacked.
Select the unzipped FOHKASU folder.
Initialize: Click the extension icon and set up your profile. Be honest.
